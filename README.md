# Caspor

This game is about a little ghost called Caspor, you are Caspor

As you play you can go through walls but you can even possess other objects allowing you to push other objects

Controls:

To move: WASD or ↑  ← ↓ →

To fly: E or Space to go up / Q or Shift to go down

To possess an object: Look at the target and press F

To return to ghost body: Q

To return to main menu: Escape (Esc)


I made this project as a 10 hour project for Hackclub seige week 9 for the theme based on Spooooky
